# java-web-gradle-spring-jsf-postgres-simple

## Description
A jsf springboot java gradle build,
that connects to postgres database.

## Tech stack
- springboot
  - web
- jsf
- gradle
- bootstrap
- jquery
- dataTable

## Docker stack
- postgres
- gradle:jdk11

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
